# KeysMapInfo4 — Changelog

## 1.3.0 (2026-06-28)

### Добавлено
- **Цветовое кодирование ключей** (`colorCoding` в config.json): фон в инвентаре/схроне по карте (из ColorCodedKeys / KeysMapInfo 1.2).
- Помеченные ключи остаются жёлтыми (`changeMarkedKeysBackground: false`).
- Мусорные ключи — чёрные (`junkKeys`).

## 1.2.3 (2026-06-28)

### Изменено
- Обработка только указанных языков (`locales` в config.json). По умолчанию: `["ru"]`.
- Вместо всех 25 языков сервера мод трогает только нужные — быстрее старт.

## 1.2.2 (2026-06-28)

### Исправлено
- Зависание при старте: локализация читается через `LocaleService.TryGetLocaleDb`, а не через `LazyLoad.Value` на `Locales.Global` (SPT 4 явно не рекомендует прямой доступ).
- Лог прогресса: число языков и обновлённых строк локализации.

## 1.2.1 (2026-06-28)

### Исправлено
- Загрузка `config.json` в формате camelCase (наследие SPT 3): словарь `maps` и `keyTypes` больше не теряются при десериализации SPT 4.
- Проверка типа ключа через `TemplateItem.Parent` вместо рефлексии по `_parent`.
- Логи: предупреждение при 0 обработанных ключах вместо ложного «успешно загружен».
